### Features
- [x] Users routes (GET, POST, PUT, DELETE)<br> 
- [x] Admin routes (GET, POST, PUT, DELETE)<br>
- [x] Automated testing<br>
- [x] Middlewares<br>
- [x] Postgres database<br>
